# FastAPI Social Media API

A backend REST API built with Python and FastAPI while following a comprehensive FastAPI backend development course.

The project provides a social-media-style API where users can register, log in, create posts, update or delete their own posts, and vote on posts.

## Features

* User registration and password hashing
* JWT-based authentication and protected routes
* Create, read, update, and delete posts
* Authorization: users can update or delete only their own posts
* Post voting / liking system
* PostgreSQL database integration
* SQLAlchemy ORM and database relationships
* Pydantic request validation and response schemas
* Alembic database migrations
* Environment variable configuration
* Automated API tests using Pytest
* CI/CD workflow using GitHub Actions
* Docker and Docker Compose setup

## Tech Stack

* Python
* FastAPI
* PostgreSQL
* SQLAlchemy
* Pydantic
* Alembic
* JWT Authentication
* Pytest
* Docker
* GitHub Actions

## API Documentation
